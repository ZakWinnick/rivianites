# Electro-Terrestrials

An Area 51 road trip event site for electric vehicle enthusiasts.

**Live site:** [area51.rivianites.online](https://area51.rivianites.online)

## About

A three-day EV adventure through Nevada's Extraterrestrial Highway, featuring:
- Las Vegas meetup and caravan
- Camping at Pickett's RV Park in Alamo
- Drive to Rachel via NV-375
- Area 51 gates exploration
- Stargazing at Black Mailbox
- EV charging info throughout

## Tech Stack

- Jekyll (GitHub Pages)
- Vanilla CSS with `prefers-color-scheme` for automatic light/dark mode
- Tinylytics analytics

## Development

```bash
bundle install
bundle exec jekyll serve
```
